Skill architecture

Operators + Lenses + Artifacts + Evidence

`$phd` 把研究能力拆成四层：做什么、怎么看、产出什么、如何保持可靠。

`$phd` separates research capability into what to do, how to look, what to produce, and how to stay reliable.

Agent 应如何使用

  1. 判断研究意图，并选择一个或多个模式。
  2. 只选择当前任务需要的 operators 和 lenses。
  3. 选择适合目标的 output artifact。
  4. 对重要 claim 应用 evidence discipline。
  5. 当需要长期产出时，创建或 enrich HTML research site。

How An Agent Should Use It

  1. Infer the research intent and choose one or more modes.
  2. Select only the operators and lenses needed for the task.
  3. Choose an output artifact that fits the intent.
  4. Apply evidence discipline to important claims.
  5. Create or enrich an HTML research site when durable output matters.
这个架构应通过真实研究任务继续演化。This architecture should evolve through real research tasks.
