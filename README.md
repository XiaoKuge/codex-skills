# codex-skills

A collection of composable skills for [Codex](https://openai.com/codex/).

## Skills

### `phd`

A composable research cognition system for systematic research thinking over a
concept, topic, field, question, corpus, paper set, or complex information
bundle. It supports collecting and organizing background information, surveying
related work, building concise conceptual frameworks, summarizing and
abstracting complex material, structuring analysis across multiple lenses
(spatial, temporal, causal, historical, actor, method, and more), and
generating insights, hypotheses, gaps, and research directions.

The default durable output is a continuously enriched static HTML research
site. The skill is composable: choose the research operators and lenses needed
for a task rather than following one fixed workflow.

See [`phd/SKILL.md`](phd/SKILL.md) for the full skill definition and the
[`phd/references/`](phd/references) directory for operators, lenses, output
patterns, and evidence discipline.
